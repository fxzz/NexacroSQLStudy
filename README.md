```
1. 기능:
사용자는 각 부서별로 평균 급여를 조회할 수 있어야 합니다.
조회 시에는 부서 목록을 선택하고, 선택한 부서의 평균 급여가 표시되어야 합니다.

2. 화면 요소:

3. 동작:
사용자가 부서 목록에서 특정 부서를 선택하면, 선택된 부서의 평균 급여가 화면에 표시됩니다.
```
